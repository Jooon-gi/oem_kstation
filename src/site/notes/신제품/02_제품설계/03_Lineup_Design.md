# Lineup Design — 라인업 포지셔닝 설계

> 목적: 3개 라인의 시장 포지셔닝, 고민 유형 트랙 연결, 커뮤니케이션 방향 설계
> 상태: [Draft]
> 최종 수정: 2026-04-10

---

## 라인업 확정 사항

- 라인 수: **3개 확정** (FINE / SUB-Q / DEEP)
- 성분: HA 24mg/mL, Lidocaine 3mg/mL (전 라인 동일)
- 제조 기술: MCO Tech (Minimising Cross-linking agent by Optimising reaction) — BDDE 최소화
- MOQ: **3,000개** (라인별 또는 총합 여부 → OEM 확인 필요)
- 목표 제작단가: **7,000-8,000 KRW/개**
- 소비자가: **$12-15** (⏳ 예정)

---

## 라인업 포지셔닝 맵

```
표층 / 섬세한 볼륨
        │
   [ FINE ]
   주입: 진피 표층
   적응증: 잔주름, fine line, 섬세한 볼륨 보정
   G' 추정: 20-100 Pa
   지속: 최대 9개월 (개인차 있음)
        │
   [ SUB-Q ]
   주입: 피하 지방층
   적응증: 중간 볼륨, 윤곽 보정, 팔자·미드페이스
   G' 추정: 101-200 Pa
   지속: 최대 12개월 (개인차 있음)
        │
   [ DEEP ]
   주입: 골막 상부 (Deep plane)
   적응증: 구조적 리프팅, 대용량 볼륨, 안면 윤곽
   G' 추정: 201-300 Pa
   지속: 최대 12개월 (개인차 있음)
        │
구조 지지 / 높은 가교 강도
```

> [!info] G' 레퍼런스 범위 해석
> 위 G' 추정값은 LINE FILL(AQUABANK) 공개 자료 기반이다. 실제 케이필<가제> Lot 수치는 OEM COA 수령 후 교체한다.
> G' 수치는 OEM이 공유하는 경우에만 제품 페이지에 표기한다.

## 라인별 포지셔닝 방향

### 케이필<가제> FINE — Track A 주력

- **주입 깊이:** 진피 표층
- **주 고민 유형:** 잔주름 / 표층 fine line / 섬세한 볼륨 보정
- **포지셔닝:** 균일한 입자 분포, Tyndall 리스크 최소화
- **핵심 메시지:** "Fine lines and subtle volume — uniform particle distribution for superficial placement."
- **타겟 사용 케이스:** 눈 밑 fine line, 이마 잔주름, 입술 경계 정리, 표층 볼륨 미세 보정
- **지속 기간:** 최대 9개월 (개인차 있음)
- **G':** 20-100 Pa (OEM 실측값 확인 예정)

---

### 케이필<가제> SUB-Q — Track B 주력

- **주입 깊이:** 피하 지방층
- **주 고민 유형:** 중간 볼륨 부족 / 미드페이스 볼륨 / 윤곽 보정
- **포지셔닝:** 범용 볼륨 추가, 자연스러운 윤곽
- **핵심 메시지:** "Mid-volume and contour — subcutaneous placement for fuller, defined structure."
- **타겟 사용 케이스:** 미드페이스 볼륨, 팔자주름, 입술 볼륨, 관자놀이, 눈 밑 볼륨
- **지속 기간:** 최대 12개월 (개인차 있음)
- **G':** 101-200 Pa (OEM 실측값 확인 예정)

---

### 케이필<가제> DEEP — Track C 주력

- **주입 깊이:** 골막 상부 (Deep plane)
- **주 고민 유형:** 구조적 리프팅 / 대용량 볼륨 추가 / 안면 윤곽 재설계
- **포지셔닝:** 골막 위 구조적 지지, 높은 가교 강도
- **핵심 메시지:** "Structural lift and high-volume augmentation — supraperiosteal placement for lasting definition."
- **타겟 사용 케이스:** 볼 볼륨, 턱 윤곽, 광대 구조, 이마 골막 위 볼륨
- **지속 기간:** 최대 12개월 (개인차 있음)
- **G':** 201-300 Pa (OEM 실측값 확인 예정)

> [!tip] 기획 의도 — DEEP이 가장 강한 가교를 갖는 이유
> 골막 위 배치는 외부 압력을 직접 받는 부위다. 형태 유지를 위해 가장 높은 G'가 요구된다.
> SUB-Q보다 깊은 배치 → 혈관/신경 주변 시술 → 시술자 역량 중요. 마케팅에서 시술 깊이를 자세히 묘사하지 않는다.
