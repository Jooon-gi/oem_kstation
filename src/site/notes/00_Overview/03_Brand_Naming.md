# 03. Brand Naming — 브랜드명 선정

> 목적: 1차 출시 브랜드명 후보 정리 및 클라이언트 컨펌 프로세스
> 최종 수정: 2026-04-10
> 상태: [Draft — 클라이언트 컨펌 대기]

---

## 선정 기준

| 기준          | 설명                                                               |
| ----------- | ---------------------------------------------------------------- |
| **발음 가용성**  | 영어권 + 비영어권(히스패닉, 아시아계, 중동계) 모두 직관적으로 읽힐 것                        |
| **어원 투명성**  | 어원 또는 연상 이미지가 브랜드 철학과 연결될 것                                      |
| **클레임 중립성** | "FDA approved", "clinically proven" 같은 규제 위반 클레임을 함의하지 않을 것      |
| **구별성**     | 기존 유통 브랜드(Revolax, Neuramis, Juvederm 등)와 혼동 없을 것. 기존 동명 상품 없을 것 |
| **확장성**     | FINE / DEEP / SUB-Q 라인명과 자연스럽게 결합할 것                             |

---

## 후보 비교 요약

| 브랜드명        | 연상 이미지       | 핵심 맥락          | 문화 범용성 | 상표 위험도      |
| ----------- | ------------ | -------------- | ------ | ----------- |
| Vera Fill   | 신뢰, 검증됨      | 안심, 인증 배경      | ★★★★   | ✅ 낮음        |
| Essen Fill  | 본질, 정수       | 군더더기 없음        | ★★★    | ✅ 낮음        |
| Vita Fill   | 생명력, 활력      | 활성·생기 부여       | ★★★★   | ✅ 낮음        |
| Sylva Fill  | 자연, 숲, 재생    | 자연 유래·재생 이미지   | ★★★    | ✅ 낮음        |
| Purä Fill   | 순수, 숨김 없음    | 스펙 투명성         | ★★★★   | ⚠️ 중간       |
| Nectar Fill | 촉촉함, 영양      | 꿀처럼 풍부한 수분·영양  | ★★★★   | ⚠️ 중간       |

> 상표 위험도는 웹 검색 기반 조사 결과. 최종 출원 전 USPTO TESS 전문 검색 필수.

## 후보군 상세

### 1. Vera Fill (베라 필)

> 라틴어 **vera** — Verified, 진실한, 진짜의, 검증된

| 항목 | 내용 |
| --- | --- |
| **핵심 맥락** | "인증되어 있고, 안심할 수 있다." 제조사 ISO 13485 + CE Mark 이력을 배경으로 신뢰성을 함의. |
| **라인 결합** | VERAFIL FINE / VERAFIL DEEP / VERAFIL SUB-Q |
| **장점** | 의료·미용 카테고리에서 "Vera"는 검증·진정성 연상이 강함. 발음이 짧고 명확. |
| **상표 조사 결과** | 독립 필러 브랜드 미확인. ✅ 사용 가능 가능성 높음 — USPTO 최종 확인 필요. |

---

### 2. Essen Fill (에센 필)

> 영어 **essential** + 독일어 **Essen** (본질) 합성

| 항목 | 내용 |
| --- | --- |
| **핵심 맥락** | 꼭 필요한 성분만, 군더더기 없이. "브랜드 프리미엄 없이 정수만 담았다." 고관여층 세그먼트에 특히 유효. |
| **라인 결합** | ESSENFIL FINE / ESSENFIL DEEP / ESSENFIL SUB-Q |
| **장점** | "Essential"을 연상시키며 스펙 지향 유저에게 직관적. 북유럽·미니멀 감성. |
| **주의** | "Essen"이 영어권에서 생소할 수 있음. 발음 교육 없이 노출 시 오독 가능. |
| **상표 조사 결과** | 독립 필러 브랜드 미확인. ✅ 사용 가능 가능성 높음 — USPTO 최종 확인 필요. |

---

### 3. Vita Fill (비타 필)

> 라틴어 **vita** — 생명, 생명력, 활력

| 항목 | 내용 |
| --- | --- |
| **핵심 맥락** | "생명력을 부여한다." 피부에 생기와 활력을 되돌린다는 이미지. |
| **라인 결합** | VITAFIL FINE / VITAFIL DEEP / VITAFIL SUB-Q |
| **장점** | "Vita" 접두어는 비타민·활력 연상으로 뷰티 카테고리에서 긍정적 인식. 발음 직관적. |
| **주의** | "Vita" 접두어를 쓰는 미용 클리닉이 다수 존재. 필러 제품 직접 충돌은 없음. |
| **상표 조사 결과** | "Vita Fill" 정확 일치 브랜드 미확인. ✅ 사용 가능 가능성 높음 — USPTO 최종 확인 필요. |

---

### 4. Sylva Fill (실바 필)

> 라틴어 **silva/sylva** — 숲, 자연

| 항목 | 내용 |
| --- | --- |
| **핵심 맥락** | 숲처럼 자연에서 온 성분, 자연스러운 재생. Non-animal origin HA 소구와 연결 가능. |
| **라인 결합** | SYLVAFIL FINE / SYLVAFIL DEEP / SYLVAFIL SUB-Q |
| **장점** | 자연 유래·클린뷰티 감성과 연결. 발음이 부드럽고 독특함. |
| **주의** | 2024년 런던 스킨케어 브랜드 "Sylva"가 존재하나 필러 카테고리와 직접 충돌 없음. |
| **상표 조사 결과** | 필러 분야 직접 충돌 미확인. ✅ 사용 가능 가능성 높음 — USPTO 최종 확인 필요. |

---

### 5. Purä Fill

> 라틴어/이탈리아어 **pura** — 순수한, 깨끗한

| 항목 | 내용 |
| --- | --- |
| **핵심 맥락** | 성분 순수성, 숨긴 것 없음. "단순하고 깨끗한 조합." 스펙 투명성 메시지와 직결. |
| **라인 결합** | PURAFIL FINE / PURAFIL DEEP / PURAFIL SUB-Q |
| **장점** | "Pure"는 뷰티·더마 카테고리에서 범용 긍정 인식. 발음이 직관적. |
| **⚠️ 주의** | "Pura" 계열 클리닉·스파 상호 다수 존재 (Pura Medical Aesthetics, Pura Vida 등). 필러 제품 브랜드 직접 충돌은 미확인이나 USPTO 조사 필수. |

---

### 6. Nectar Fill (넥타 필)

> 영어 **nectar** — 꽃의 꿀, 신들의 음료, 달콤하고 풍부한 영양

| 항목           | 내용                                                                                        |
| ------------ | ----------------------------------------------------------------------------------------- |
| **핵심 맥락**    | "꿀처럼 촉촉한 영양." 피부에 풍부한 수분과 영양을 채운다는 이미지.                                                   |
| **라인 결합**    | NECTARFIL FINE / NECTARFIL DEEP / NECTARFIL SUB-Q                                         |
| **장점**       | 감각적이고 풍부한 이미지. 뷰티·스킨케어에서 "꿀" 성분 연상과 궁합 좋음.                                                |
| **⚠️ 주의**    | "Nectar Aesthetics", "Nectar Body Solutions" 등 필러 시술을 제공하는 미용 클리닉이 미국 내 다수 존재. 혼동 가능성 있음. |
| **상표 조사 결과** | "Nectar Fill" 정확 일치 제품 미확인. ⚠️ 단, "Nectar" 브랜드 미용 클리닉이 많아 USPTO 조사 필수.                    |

## 클라이언트 컨펌 프로세스

```
1. 담당자 → 클라이언트: 후보군 + 요약표 공유
2. 클라이언트: 1-2순위 선정 또는 별도 의견 제시
3. 필요 시 상표 검색 (미국 USPTO, 캐나다 CIPO)
4. 클라이언트 최종 확정
5. 이 문서 상태 → [Confirmed] 변경
6. 전체 문서 케이필<가제> → 확정명으로 일괄 치환
7. 폴더명 및 파일명 변경 (필요 시)
```

### 확정 후 처리 항목

- [ ] 전체 문서 `케이필<가제>` → 확정 브랜드명 일괄 치환
- [ ] 패키징 디자인 파일에 브랜드명 반영
- [ ] 소셜 계정 핸들 선점 (TikTok / Instagram / Reddit)
- [ ] 미국 USPTO 상표 검색 (`케이필<가제>` + "Fill")
- [ ] OEM 패키지 라벨 디자인 시 확정명 적용
