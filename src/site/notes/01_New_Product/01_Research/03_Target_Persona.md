# Target Persona — 핵심 구매자 프로파일

> 목적: 스펙 결정 및 마케팅 언어 설계 시 참조하는 구매자 페르소나
> 최종 업데이트: 2026-04-10
> **두 축: 경험 수준(뉴비/고관여층) × 고민 유형(Track A-C)**

---

## 경험 수준 축

### Persona 1 — "The Newbie" (뉴비)
한줄요약: **시술 관련 입문용**

| 항목     | 내용                                                  |
| ------ | --------------------------------------------------- |
| 연령     | 25-38세                                              |
| 지역     | 북미 (미국, 캐나다)                                        |
| 배경     | 스킨케어 고관여. 더마롤러, 비타민C 세럼 등 적극 사용. 필러는 미경험 또는 초기 탐색 중 |
| 정보 수준  | 스킨케어 성분은 알지만 HA 필러 스펙(G', Buffer)은 낯섦               |
| 진입 장벽  | 가격("클리닉은 너무 비싸") + 정보("뭘 골라야 할지 모르겠다")              |
| 구매 동기  | "**시도해 보지 못했던 시술 분야로 이번기회에 진입**" - 가격부담이 없는 입문용     |
| 설득 포인트 | 낮은 가격 + "내 고민에 맞는 라인 골라줌" 안내                        |
| 반발 트리거 | 어려운 전문 용어, 복잡한 제품 선택, 비싼 가격                         |
| 주요 채널  | TikTok (발견) → Instagram (정착)                        |
| 콘텐츠 형식 | 비포/애프터, "처음 해봤어요" 후기, 고민 유형별 라인 가이드                 |

---

### Persona 2 — "The Veteran" (고관여층)
한줄요약: **같은 성분에서 가성비 최대치로**

| 항목     | 내용                                                  |
| ------ | --------------------------------------------------- |
| 연령     | 28-45세                                              |
| 지역     | 북미 (미국, 캐나다)                                        |
| 배경     | 클리닉 시술 경험 다수 또는 DIY 경험. 성분과 시술 이해도 높음               |
| 정보 수준  | HA, G', 가교 개념 이해. Reddit r/DIYaesthetics 등 커뮤니티 활동  |
| 구매 동기  | "**어차피 필러는 다 비슷하고, 더 싸게 사고 싶다**" - 가성비 + 쟁여두기       |
| 설득 포인트 | 성분 직설 (HA 24mg/mL, Lido 3mg/mL) + 가격 비교 + 제조사 인증 근거 |
| 반발 트리거 | 신상 브랜드 신뢰 문제                                        |
| 주요 채널  | Reddit (신뢰 구축) → Instagram (구매 전환)                  |
| 콘텐츠 형식 | 성분 비교, 벌크 구매 논리, Q&A 대응                             |

> [!tip] 기획 의도 — 뉴비와 고관여층이 서로를 강화한다
> 고관여층이 Reddit에서 케이필<가제>을 긍정적으로 언급하면 → 뉴비가 발견하고 신뢰해서 진입
> 뉴비가 TikTok/Instagram에서 구매 후기를 올리면 → 고관여층이 검토하고 전환
> 두 타겟은 채널이 다르고 메시지가 다르지만, 결과적으로 서로의 신뢰를 쌓아준다.

---

## 고민 유형 축 (Track A-C)

> 동일 페르소나(뉴비/고관여층) 내에서 시술 목적과 부위에 따라 콘텐츠와 추천 라인이 달라진다.
> 초기에는 전 트랙을 동시 배포하고, 반응 데이터 기반으로 집중 트랙을 결정한다.

| 트랙 | 고민 유형 | 주 시술 부위 | 권장 라인 | 핵심 인사이트 |
| --- | --- | --- | --- | --- |
| **Track A** | 잔주름 / 섬세한 볼륨 보정 | 눈 밑, 이마, 표층 fine line | FINE | 가장 얕은 주입. Tyndall 리스크 인식 높음. 섬세한 균일 입자 소구. |
| **Track B** | 중간 볼륨 / 윤곽 보정 | 미드페이스, 팔자주름, 입술, 관자놀이 | SUB-Q | 가장 넓은 적응증. 뉴비 진입 빈도 높음. 자연스러운 볼륨·윤곽 소구. |
| **Track C** | 구조적 리프팅 / 대용량 볼륨 | 볼, 턱, 광대, 이마 | DEEP | 가장 깊은 배치. 고관여층 비중 높음. G' 수치와 지속력 소구. |
