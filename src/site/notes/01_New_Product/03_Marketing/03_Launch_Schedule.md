# Launch Sequence — 런칭 타임라인

> 최종 업데이트: 2026-04-10
> **1차 출시 목표: 2026-07**
> 전략 기준: 가격 경쟁력 + 고민 유형별 콘텐츠 분화 → 반응 트랙 집중

---

## 전체 흐름

```
[Step 1] 디자인            4월 중순 ~ 말   패키징 / 라벨 / 브랜드 디자인 완료
[Step 2] 제조              5월 ~ 6월 중순  OEM 발주 → 제조 (1.5-2개월)
[Step 3] 콘텐츠 제작         5월 ~ 6월       제조 기간 병행 — 트랙별 콘텐츠 제작
[Step 4] 론칭              7월 초          제품 입고 확인 후 전 트랙 동시 배포
[Step 5] 신호 수집          7월 1-6주       트랙별 반응 측정
[Step 6] 집중 전환          8월~            반응 높은 트랙으로 리소스 집중
```

---

## Step 1 — 디자인 (4월 중순 ~ 말)

**완료 목표: 4월 말**

- [ ] 브랜드명 클라이언트 최종 컨펌 → `00_Overview/03_브랜드_네이밍.md`
- [ ] 패키징 디자인 방향 확정 (브랜드 컬러, 라인별 구분, 고민 유형 가이드 텍스트 포함)
- [ ] 시린지 라벨 / 박스 / Tyvek 파우치 디자인 완료
- [ ] OEM에 디자인 파일 전달 → 발주 준비

---

## Step 2 — 제조 (5월 ~ 6월 중순)

**제조 기간: 약 1.5-2개월**

- [ ] OEM 발주 확정 (MOQ, 단가, 라인별 수량)
- [ ] 제조 착수 확인
- [ ] 중간 점검 — Lot 번호 / 생산 일정 확인
- [ ] COA 수령 및 검수 → `04_Operations/02_COA_체크리스트.md`
- [ ] 입고 일정 확인 → 론칭 일정 역산

---

## Step 3 — 콘텐츠 제작 (5월 ~ 6월, 제조 기간 병행)

**제조 기간 중 병행 진행 — 론칭 전 완료 목표**

> **콘텐츠 방향**
> **Reddit 코멘트 작성 업무**에서 축적한 노하우 활용. 커뮤니티에서 반복적으로 등장하는 고민·질문을 직접 수집하고, 이를 설명해주는 방식으로 영상 제작.
> 촬영 방식은 두 가지 방향으로 검토 중:
> - 얼굴이 드러나지 않는 형태로 직접 촬영
> - 영상에 나올 스피커(출연자) 섭외

**제작할 콘텐츠 유닛 (최소)**

| 트랙                           | 뉴비용               | 고관여층용            | 플랫폼             |
| ---------------------------- | ----------------- | ---------------- | --------------- |
| Track A (잔주름 / 표층 볼륨 → FINE) | 고민 유형 가이드 영상 1편   | Reddit 게시 텍스트 1편 | TikTok / Reddit |
| Track B (중간 볼륨 / 윤곽 → SUB-Q) | 고민 유형 가이드 영상 1편   | Reddit 게시 텍스트 1편 | TikTok / Reddit |
| Track C (구조적 리프팅 → DEEP)     | 고민 유형 가이드 영상 1편   | Reddit 게시 텍스트 1편 | TikTok / Reddit |
| 공통 가격 앵커                     | 클리닉 vs 케이필<가제> 비교 | 성분 직설 포스팅        | 전 채널            |

**콘텐츠 공통 요소**
- 각 영상/포스팅에 고민 유형(잔주름 / 볼륨 / 리프팅) 명시
- 제품 페이지 링크 포함
- 가격($12-15/⏳ 예정) 명시
- 의료 조언 없음 — 가이드 형태로만

**채널 세팅 (론칭 전 완료)**
- [ ] Instagram 계정 + 브랜드 프로필 세팅
- [ ] TikTok 계정 개설
- [ ] 제품 페이지 URL 구조 확정 (트랙별 랜딩 또는 단일 페이지)
- [ ] Reddit 계정 준비 (J.S. 페르소나 — 기존 Kstation_Reddit 프로토콜 참조)

---

## Step 4 — 론칭 (7월 초)

- [ ] 제품 입고 및 COA 최종 확인
- [ ] 제품 페이지 오픈
- [ ] 전 트랙 콘텐츠 동시 배포 (TikTok, Instagram, Reddit)
- [ ] 첫 주문 배치 출고 확인
- [ ] J.S. 페르소나 Reddit "발견" 구조 댓글 시작
      (직접 광고 아닌 자연스러운 언급 — Kstation_Reddit 프로토콜 준수)

---

## Step 5 — 신호 수집 (7월, 론칭 후 1-6주)

**측정 지표**

| 신호 유형       | 측정 항목             | 기록 위치                               |
| ----------- | ----------------- | ----------------------------------- |
| 1차 (인게이지먼트) | 저장, 공유, 댓글, DM    | `05_Logs/02_커뮤니티_신호_로그.md` |
| 2차 (관심)     | 제품 페이지 클릭, 체류 시간  | 제품 페이지 분석 도구                        |
| 전환          | 첫 구매 수, 트랙별 구매 비율 | 별도 집계                               |
| 커뮤니티 반응     | Reddit 언급, 질문 유형  | `05_Logs/02_커뮤니티_신호_로그.md` |

**판단 기준 (6주 후)**
- 특정 트랙에서 저장률/공유율이 2배 이상 높으면 → 그 트랙으로 집중 전환
- 뉴비/고관여층 중 어느 쪽 전환율이 높은지 확인
- 완판 여부 → 추가 생산 vs. 브랜딩 조정 결정

---

## Step 6 — 집중 전환 (8월~)

- [ ] 반응 높은 트랙 콘텐츠 증산
- [ ] 반응 낮은 트랙 일시 중단 또는 메시지 수정
- [ ] 완판 시: 추가 생산 발주 + 반응 트랙 기반 다음 방향 결정
- [ ] 미완판 시: 커뮤니티 반응 데이터 분석 → 브랜딩/타겟 재조정
