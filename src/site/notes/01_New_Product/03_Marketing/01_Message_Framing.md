# Messaging Framework — 브랜드 메시지 구조

> 최종 업데이트: 2026-04-10
> **기준 문서: `00_Overview/01_제품전략_핵심.md`**
> 모든 메시지는 두 축(경험 수준 × 고민 유형)을 기준으로 분기된다.

---

## 브랜드 메시지 원칙

1. **가격을 숨기지 않는다.** $12-15(⏳ 예정)는 강점이다. 앞에 내세운다.
2. **성분은 있는 그대로만 말한다.** HA 24mg/mL, Lidocaine 3mg/mL.
3. **고민 유형으로 말한다.** "모든 분들에게 맞아요"가 아니라 "이 고민엔 이 라인이 맞아요."
4. **인증 없는 것은 인증 있는 척하지 않는다.** 제조사 기준만 언급.

---

## 메시지 레벨

```
[Level 1] 브랜드 약속 — 즉시 사용 가능
"Accessible price. Your concern, your guide."
"$12-15. No brand tax. Just HA."

[Level 2] 경험 수준별 메시지 — 즉시 사용 가능
→ 뉴비:  "클리닉 가격의 1/20. 처음 시작하는 분들을 위한 라인 가이드 포함."
→ 고관여층: "HA 24mg/mL. 성분은 아시잖아요. 가격만 보세요."

[Level 3] 고민 유형별 메시지 — 트랙별 콘텐츠 제작 시 작성
→ Track A (잔주름 / 표층 볼륨): "Fine lines and subtle volume — FINE으로 시작하세요."
→ Track B (중간 볼륨 / 윤곽):   "Mid-face volume and contour — SUB-Q가 맞는 선택."
→ Track C (구조적 리프팅):      "Structural lift and high-volume — DEEP for lasting definition."
```

---

## 뉴비 메시지 상세

### 핵심 인사이트
스킨케어에 이미 투자하고 있지만 필러는 "너무 비싸거나 무서운 것"으로 인식.
가격 장벽과 정보 장벽, 두 가지를 동시에 낮추는 것이 진입 유도의 핵심.

### 메시지 방향
- **비용 재프레이밍:** "매달 세럼에 쓰는 돈이면 케이필<가제> 한 개 남는다."
- **안내 강조:** "내 고민 유형 확인하면 어떤 라인이 맞는지 알려드림."
- **진입 장벽 제거:** "처음 해보는 분들도 볼 수 있는 라인 가이드 포함."

### 사용 가능 카피 초안
> "Clinic prices start at $400.
>  케이필<가제> starts at $12(⏳ 예정).
>  Same HA formulation. Your concern type guide included."

> "Been spending on serums? This is the next step.
>  Tell us your concern → we'll guide you to the right line."

### 금지 방향
- 시술 방법, 깊이, 테크닉 구체적 묘사 (의료 클레임 인접)
- "안전하다", "부작용 없다" 계열 표현

## 고관여층 메시지 상세

### 핵심 인사이트
이미 필러를 알고 쓰고 있음. 성분 검증은 본인이 할 수 있음.
필요한 건 "이 성분이 이 가격에 맞는가"에 대한 확신.

### 메시지 방향
- **성분 직설:** "HA 24mg/mL. Lidocaine 3mg/mL. 표준 구성."
- **기술 소구:** "MCO Tech — BDDE 최소화. Non-animal origin."
- **가격 직설:** "Revolax 한 개 가격으로 세 개."
- **쟁여두기 논리:** "같은 성분 클래스, 1/3 가격."

### 사용 가능 카피 초안
> "HA 24mg/mL. Lidocaine 3mg/mL.
>  You already know what this means.
>  $12-15. Stock accordingly."

> "Same formulation class as what you've been using.
>  A third of the price."

##  사용 금지 문구

| 금지 문구                           | 이유                           |
| ------------------------------- | ---------------------------- |
| "CE certified" / "FDA approved" | 인증 없음                        |
| "clinically proven"             | 근거 없음, liability             |
| "professional grade"            | 근거 없는 클레임                    |
| "safe for home use"             | 법적 리스크                       |
| "doctor-recommended"            | 허위 광고 리스크                    |
| "lasts X months"                | 효능 클레임 (개인차 있음 표기 없으면 사용 불가) |
| "PBS-buffered"                  | Buffer 타입 미확인                |
| "pH X.X"                        | pH 수치 미확인                    |
| "G' X Pa"                       | G' 수치 OEM 미확인 상태             |
| "just like what clinics use"    | 직접 비교 클레임                    |
